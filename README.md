# كوميك ترجم 💬

تطبيق ويب لترجمة الكوميكس من الإنجليزية إلى العربية باستخدام الذكاء الاصطناعي.

## النشر على Vercel (مجاناً)

### الخطوة 1 — رفع الكود على GitHub
1. اذهب إلى [github.com](https://github.com) وأنشئ حساباً (أو سجّل دخول)
2. اضغط **New repository**
3. اسمّه `comic-translate-web`
4. اضغط **Create repository**
5. في الصفحة التالية اضغط **uploading an existing file**
6. ارفع كل الملفات من هذا المجلد

### الخطوة 2 — النشر على Vercel
1. اذهب إلى [vercel.com](https://vercel.com) وسجّل دخول بـ GitHub
2. اضغط **Add New Project**
3. اختر `comic-translate-web` من قائمة المشاريع
4. اضغط **Deploy** — بدون أي إعدادات إضافية!
5. بعد دقيقة واحدة يعطيك رابط مثل: `https://comic-translate-web.vercel.app`

### الخطوة 3 — الاستخدام
افتح الرابط من أي متصفح أو آيفون، أدخل مفتاح Anthropic API وابدأ الترجمة!

## الميزات
- 🔍 كشف فقاعات الحوار (speech, thought, narration, sfx)
- 🌐 ترجمة من الإنجليزية إلى العربية
- 🖼️ رسم bounding boxes على الصورة
- 💾 تحميل النتائج JSON
- 🔑 حفظ مفتاح API تلقائياً في المتصفح
- 📱 يعمل على الآيفون والأندرويد

## المتطلبات
- مفتاح Anthropic API من [console.anthropic.com](https://console.anthropic.com)
